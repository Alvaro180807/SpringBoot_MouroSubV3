# SpringBoot_MouroSubV3

Proyecto Spring Boot para gestionar la DB de "MouroSub" de forma sencilla.

Incluye una pequeña capa con controlador, servicios, repositorios y carga inicial de datos.

## Tecnologías
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL

## Ejecución
1. Configura la base de datos en `src/main/resources/application.properties`.
2. Ejecuta la aplicación
